# AWS Lambda & S3 Lab

## Descrição
Este laboratório tem como objetivo consolidar conhecimentos em tarefas automatizadas usando AWS Lambda e S3.  

## Estrutura do Repositório
- `lambda_function/` — Contém a função Lambda e dependências.
- `s3_bucket/` — Arquivos de exemplo para upload/download no S3.
- `notes/` — Anotações e insights adquiridos durante a prática.

## Como executar
1. Configure sua AWS CLI com credenciais válidas.
2. Crie um bucket S3 na sua conta.
3. Faça deploy da função Lambda.
4. Teste o upload/download de arquivos usando o código fornecido.
